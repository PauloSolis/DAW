•URL: sus siglas Uniform Resource Locator es la referencía que usa el navegador para hallar la ubicación del algún recurso, y recuperarlo.
•MIME: Multipurpose Internet Mail extensions es una manera estandarizada the indicar el formato de un documento Ejemplo text/html o image/png.
•XML: eXtensible Markup Language es un lenguaje cuyo objetivo es definir un conjunto de reglas através de todo el internet. Su formato es leíble tanto por humano y por máquina.
•XHTML:EXtensible HyperText Markup Language es casi igual que el HTML pero, es más estricto. Es HTML definido como una aplicación XML.
•JS: También conocido como JavaScript es un lenguaje que fue creado principalmente para hacer que las páginas tuvieran más vida.
•PHP: acrónimo de Hypertext ProProcessor es un lenguaje script, a diferecnia de JS que se ejecuta en el cliente, mientras PHP en el servidor
•JSP: Con un método dinámico independiente para la creación de páginas web. Cuenta con todas las APIS de Java.
•Ajax: lenguaje que te permite hacer erramientas asincronas, es decir no necesitas relodear la página para obtener nuevos datos
